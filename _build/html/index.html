This method is used to generate and view a NaN table. It contains the number of missing values and the percentage of them for each column.

Parameters

Returns

The NaN table

dataframe

inplace_missings(df, column, method, n_neighbors=2)

This method inplaces missing values of a given table with the method wanted.

Parameters

  • df (dataframe) – The table where the missing values will be filled

  • column (str) – Column of the df to inplace missing values

  • method (str) – The method that will be used to inplace the missing values

  • n_neighbors (int) – Only used if the method chosen is KNN to inplace missings

Returns

dataframe with the missing values replaced

dataframe
